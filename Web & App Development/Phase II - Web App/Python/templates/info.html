1. Narrative

Pickup Sport is a both website and app that connect you and others with pickup sport events in Austin. There are a lot of venues for pickup sports in Austin but there is no way to find out where there is one now (or in the future) that one can join. The idea is pretty simple, search the venue or the event that you like, then, join it!
